MAC compress _MACOSX 文件夹

macOS 在压缩文件夹时，会自动生成并加入 _MACOSX 文件夹作为一些缓存，该文件夹在 MACOS 上看不到，在 windows 可以看得到。

删除方式: 如何删除 macOS 压缩包中的隐藏文件 | 一日一技 - 少数派 zip 的几行命令写入了 automator, 右键-service 运行净化 zip，等待弹窗提示即可。

Mac preview 多标签打开

system preference -> General -> Perfer tabs [选项] when opening document

选项处选择 “always”

lsof

netstat： netstat -tnl 查看开放的端口 netstat -aux 查看多少个网络连接

使用 lsof 代替 Mac OS X 中的 netstat 查看占用端口的程序 关闭监听某个端口的程序： kill -9 $(lsof -i:20002|awk '{print $2}'|sed -n 2p)
